# 🛒 E-commerce API – Flask

Este projeto é uma API REST simples de e-commerce desenvolvida em **Python com Flask**.  
Ela possui autenticação de usuários, cadastro de produtos, carrinho de compras e checkout.

O objetivo é praticar conceitos de back-end como:
- CRUD
- Autenticação
- API REST
- Banco de dados com SQLAlchemy
- Relacionamento entre tabelas

---

## 🚀 Funcionalidades

- Login e Logout de usuários  
- Cadastro de produtos  
- Listagem de produtos  
- Buscar produto por ID  
- Atualizar produto  
- Remover produto  
- Adicionar item ao carrinho  
- Remover item do carrinho  
- Visualizar carrinho  
- Checkout (limpa o carrinho)  

---

## 🧰 Tecnologias Utilizadas

- Python 3  
- Flask  
- Flask-SQLAlchemy  
- Flask-Login  
- Flask-CORS  
- SQLite
